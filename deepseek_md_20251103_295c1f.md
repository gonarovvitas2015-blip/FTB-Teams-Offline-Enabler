# FTB Teams Offline Enabler

Enables FTB Teams functionality in offline mode for Minecraft 1.21.1 with NeoForge.

## System Requirements
- **Minecraft**: 1.21.1
- **Loader**: NeoForge
- **Java**: 17 or higher
- **Required**: FTB Teams mod

## Features
- Allows FTB Teams to work in offline/lan mode
- Compatible with NeoForge 1.21.1
- Java 17+ compatible
